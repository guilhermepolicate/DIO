# Xbox Game Pass Subscriptions – Projeto de Análise de Dados

## Visão Geral
Este projeto apresenta uma **dashboard interativo** desenvolvido em **Microsoft Excel**.

O arquivo combina uma **base de dados estruturada** com um **dashboard executivo**.

## Objetivos do Projeto
- Analisar o volume de assinaturas do Xbox Game Pass
- Avaliar a adesão dos clientes a produtos adicionais (EA Play Season Pass e Minecraft Season Pass)
- Comparar os tipos de assinatura (Monthly, Quarterly e Annual)

## Como Utilizar
1. Abra o arquivo `Projeto.xlsx` no Microsoft Excel
2. Acesse a aba principal do dashboard
3. Utilize os filtros de **Subscription Type** para segmentar a análise
